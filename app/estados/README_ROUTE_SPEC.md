# /app/estados route spec

## Goal
A national index of CAAs with:
- region grouping
- search by UF / estado / CAA name
- lawyer count
- site / Instagram / Facebook links
- app availability label
- short PT-BR service summary
- visual badge when a row remains in the QA queue

## Main dataset
- `public/data/estados_cards.json`

## Card fields
- uf
- estado
- regiao
- region_color
- caa_nome
- site_oficial
- instagram_url
- facebook_url
- total_advs
- status_app_ptbr
- resumo_curto_ptbr
- has_review_flag
